# Plantiful

[Expo Documentation](https://docs.expo.dev/)

## Setup

```sh
./setup-<platform>.sh
```

Or just install [yarn](https://classic.yarnpkg.com/lang/en/docs/install) and [Node.js](https://nodejs.org), then run `yarn install` and start the app.

### Start

```sh
yarn start
```

### Install new package

```sh
yarn add <package-name>
```
